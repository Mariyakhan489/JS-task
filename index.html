For Loop and While Loop with examples.

For Loop

No. Task Description Example Code
#1 Print Numbers 1-10 Displays numbers from 1 to 10 for(let i=1;i <= 10;i++){console.log(i);}
#2 Print all Even Numbers 1-20 Displays only even numbers: 2,4,6,... for(let i=2;i <= 20;i += 2){console.log(i);}
#3 Multiplication Table of 5 Displays 5 × 1 = 5 for(let i=1;i <= 10;i++){console.log(5*i);}
#4 Print Reverse Order Numbers 10-1 Displays numbers 10, 9, 8 … 1 for(let i=10;i >= 1;i--){console.log(i);}
